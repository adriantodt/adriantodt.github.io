Things I made

Aru! (GitHub – Discord)

Aru is a Discord bot made in Kotlin that I created after taking a break from Mantaro, another Discord bot.

Since the Music bot apocalypse, Aru was open-sourced and is running with a private build which doesn't include music anymore.

Psi (GitHub)

Psi is a Catnip-based Discord bot Framework that I wrote in Kotlin.

It features a command framework with Kodein-based dependency injection and modular argument parsing

Things I helped

Mantaro (GitHub – Website)

Mantaro is a Discord bot made with Kodehawa and it got big.

I'm not an active developer anymore, but I contributed a lot of code during the 3 years of active development.

Kaiper (GitHub)

Kaiper is a functional scripting language for the Java programming language, which supports embedded execution in the runtime using the interpreter or the custom bytecode VM.

This project was started by Avarel and I contributed with a bytecode and a stack-based virtual machine.

RethinkDB Java Driver (GitHub)

I'm currently the main developer for the RethinkDB Java Driver, and published versions 2.4.0 onward of the driver.

Catnip (GitHub)

Catnip is a Discord API wrapper in Java, fully async/reactive, built on top of RxJava.

I made some contributions to getting version 2.0 released, such as implementing the event dispatcher.
